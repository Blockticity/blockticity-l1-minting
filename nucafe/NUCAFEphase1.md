## Methodology

### Minting Process
1. **Data Source:** NuCafe Coffee Worksheet (`NuCafe Coffee Worksheet - Blockcity(1).xlsx`) containing 42 farm records with GPS coordinates, hectares, and region data
2. **Builder Tool:** Blockticity COA Builder (GVE) at `localhost:3000/coa-builder.html` with the Farm License template
3. **Template Fields:** Farm Name, Farmer ID, Hectares, Region, Country, Harvest Period, Deforestation Risk, Shipment Date, Coffee Type, Total Bags, Green Coffee (kg), Contract Ref, Lot No. / ICO No.
4. **Certificate Title:** NUCAFE COFFEE LICENSE
5. **Issuer Preset:** Uganda Coffee Development Authority - NUCAFE (auto-filled)
6. **Each COA includes:** NUCAFE logo, geospatial map with GPS coordinates, Blockticity verification seal, QR code, GVE fingerprint, and verification pattern

### On-Chain Verification
- All 42 transactions verified via `eth_getTransactionReceipt` RPC calls — 100% success (status `0x1`)
- All 42 tokens confirmed via `ownerOf()` contract call — all owned by minting wallet
- Token metadata verified via `tokenURI()` — base64-encoded JSON with full product data
- SVG images verified on S3 with embedded GPS coordinates matching spreadsheet data
